# 📧 AI Email Spam Detection

## 🔍 Overview

This project is a **Machine Learning based Email Spam Detector** that classifies messages as **Spam** or **Not Spam**.

The model is trained using **Multinomial Naive Bayes** and deployed as a **web application using Streamlit**.

## 🧠 Machine Learning Model

Algorithm used:

* Multinomial Naive Bayes

## ⚙️ Installation

Clone the repository

```

```

```
pip install -r requirements.txt
```

Run the app

```
streamlit run app.py
```
